# Localization (i18n)

The platform is strictly bilingual (Arabic/English) with full RTL support.

## No Hardcoded Strings

Hardcoded UI strings are PROHIBITED in both `apps/mobile` and `apps/admin`.
You must never write raw Arabic or English text into the JSX/TSX render tree or `Alert` dialogues.

## Centralized Translation Keys

All developers must utilize centralized translation keys from the `@uniride/core` package.

Example (Mobile):

```tsx
import { useTranslation } from '../src/hooks/useTranslation';

// Inside component:
const { t } = useTranslation();

// Wrong:
<Text>تسجيل الدخول</Text>;
Alert.alert('خطأ', 'رسالة الخطأ');

// Correct:
<Text>{t('auth.login')}</Text>;
Alert.alert(t('common.error'), t('auth.login_failed'));
```

Always verify that the keys used actually exist in the core dictionaries.
